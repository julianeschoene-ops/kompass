# KOMPASS 4.0 – Werkstatt, Kernfächer, Punkte-Auswertung, LEB

Neu in Version 4.0:
- Punkte-Eingabe im Bewertungsdialog: erreichte Punkte + Maximalpunkte berechnen automatisch rot/gelb/grün.
- Schwellen sind anpassbar: Standard rot bis 49 %, gelb bis 74 %, grün ab 75 %.
- Kernfächer Deutsch, Mathematik und Englisch als eigene Ansicht ohne Units.
- Kernfach-Kompetenzen können in der Kompetenzverwaltung pro Jahrgang angepasst/ergänzt werden.
- LEB-Texte sind konkreter: sie greifen dokumentierte Kompetenzen, Punkte-Ergebnisse und Entwicklungsbereiche auf.
- Bestehende lokale Daten bleiben erhalten; neue 4.0-Kompetenzen werden beim Start ergänzt.

Wichtig:
Vor jedem Update bitte unter Einstellungen eine Datensicherung herunterladen.
