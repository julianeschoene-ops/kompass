# KOMPASS 4.2

Statische Offline-Webanwendung für Kompetenzdokumentation, Kernfächer, Werkstattunterricht und LEB-Texte.

## Dateien
- index.html
- style.css
- data.js
- storage.js
- ui.js
- students.js
- workshop.js
- leb.js
- admin.js
- app.js

## Nutzung
Alle Dateien in dasselbe GitHub-Pages-Verzeichnis kopieren. Es wird kein Server benötigt.

## Speicherung
Die Daten werden lokal im Browser über LocalStorage gespeichert. Vor Updates unter **Einstellungen** eine Datensicherung herunterladen.
